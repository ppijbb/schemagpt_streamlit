# schemagpt_streamlit

Streamlit resume and demo apps (healthcare, LLM, RAG, etc.).

## Setup

- **With uv (recommended):** `uv sync` (or `uv lock` then `uv sync`). Requires Python 3.10+.
- **With pip:** `pip install -r requirements.txt`

Optional: Neo4j support via `uv sync --extra neo4j` or `pip install langchain-neo4j`.
